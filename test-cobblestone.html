Cobblestone Terrain Implementation Test

Test Instructions:

  1. Open the VTT application at http://localhost:3000
  2. Navigate to the Level Editor
  3. Look for "Cobblestone" in the Natural Terrain section
  4. Select the Terrain Brush tool
  5. Select Cobblestone terrain type
  6. Paint some cobblestone tiles on the grid
  7. Verify that different tile variations appear randomly
  8. Try using different brush sizes to see the randomization effect

Expected Results:

  • Cobblestone should appear in the Natural Terrain category
  • When painting cobblestone, you should see different tile variations (Cobble1.png, Cobble2.png, Cobble3.png, Cobble4.png)
  • Each tile placed should randomly select one of the 4 variations
  • Larger brush sizes should show a mix of different cobblestone tiles
  • The tiles should render as actual PNG images, not just colored squares
